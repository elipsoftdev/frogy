# Frogy.app — One-pager

One-pager digital e interactivo que presenta el modelo de negocio de **Frogy.app**, la red colaborativa de precios y ahorro. Pieza pensada para presentaciones ante aceleradoras, inversionistas y aliados comerciales.

## 🐸 Qué es Frogy.app

Frogy.app es una app colaborativa que convierte la compra diaria en una decisión más inteligente. Permite comparar precios, descubrir comercios y ofertas cercanas, reportar precios reales y crear listas de compra para ahorrar antes de gastar. Cada aporte queda compartido con la red, ayudando a que más personas sepan dónde comprar mejor.

## 🌐 Sitio en vivo

El one-pager está publicado con GitHub Pages:

**https://elipsoftdev.github.io/frogy/**

## 📄 Contenido de la pieza

- Presentación del producto y soporte dual de moneda (USD / VES).
- Roadmap de monetización por etapas (comercios destacados, planes para comercios, inteligencia de precios B2B, conversión y alianzas).
- Efecto de red: cómo cada reporte fortalece la plataforma.
- Cierre de inversión con roadmap y llamado a la acción.

## 🛠️ Detalles técnicos

- Archivo único y autónomo: `index.html` (el logo va incrustado, no requiere archivos externos).
- HTML + CSS + JavaScript sin dependencias ni build.
- Animaciones nativas; respeta `prefers-reduced-motion`.
- Responsive (escritorio y móvil).

## 🚀 Ver localmente

Basta con abrir `index.html` en cualquier navegador. No requiere servidor ni instalación.
